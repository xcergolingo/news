2023-04-07 AI news

news 2023/04/07

About: AIPlugins
Insights: ChatGPT uses AIPlugins which are OpenAPI specs under the hood. The team is working on getting available LLMs to interact with them.

About: Auto-Regressive LLMs
Insights: Current Auto-Regressive LLMs are useful as writing aids, even for medical reports. However, they are not reliable as factual information sources. Writing assistance is like driving assistance, your hands must remain on the keyboard/wheel at all times.

About: Code LLMs
Insights: Simple self-improvement of Code LLMs can significantly improve code summarization and code generation performance. The process involves pre-training and fine-tuning code LLM, generating pseudo outputs, and adding them to the original data and train for the next epoch.

About: Creativity
Insights: To be productive, creativity must operate within a highly structured framework, with only a small window for novelty. Productive creativity is like a writer’s imagination, erudite and studious, while raw creativity is what you see in your dreams, formless and chaotic.

About: Dream Stories
Insights: Dreams would not make very good stories. The best stories are 80% formulaic and leverage that last 20% to produce a fresh take on the formula.

About: Encoder for Image Customization
Insights: Taming Encoder for Zero Fine-tuning Image Customization with Text-to-Image Diffusion Models is a project that aims to customize images with text without fine-tuning. The project has an abstract, project page, and Github link.

About: Generative Novel View Synthesis
Insights: Generative Novel View Synthesis with 3D-Aware Diffusion Models is a project that aims to synthesize novel views of objects from a single image. The project has an abstract and project page.

About: HNeRV
Insights: HNeRV is a Hybrid Neural Representation for Videos. The project has an abstract, project page, and Github link.

About: Intelligence
Insights: History shows that society and people’s well-being make progress with more intelligence, better skills, literacy, education, creativity, culture, communication, and the free exchange of ideas. Intelligence, by humans and machines, is what fuels progress. Machine intelligence is a way to amplify human intelligence, just as mechanical tools amplify human physical capabilities.

About: SAM
Insights: SAM is a Segment Anything Model from FAIR. It is a foundation model for image segmentation. The project has a demo, blog, paper, code, and dataset.

About: Webinar on Agents
Insights: A webinar on Agents is scheduled, featuring Yohei Nakajima and Michael Busigin. Yohei Nakajima has been doing a lot of work on baby-AGI, while Michael Busigin is working on code agents and cognosis.

About: Large Language Models
Insights: GPT4 seems to be able to answer some of the basic ML research questions. For example, the model can correctly reason that dropout may not be beneficial in the setting of GPT3.

About: Society and Technology
Insights: The faster technology moves, the faster humans adapt to it and make it theirs. What was amazing last year becomes mundane, what was weird and gimmicky becomes practical and useful. Extreme adaptability is the hallmark of human intelligence.

About: The Stormtroopers
Insights: A tweet with the caption “The Stormtroopers, on the beach” was shared.

About: Ottoman Empire
Insights: The Ottoman Empire limited the dissemination of printed books, fearing religious and political upheaval. Consequently, it missed out on the Enlightenment and lost its dominant status in science, technology, and ultimately lost its economic and military influence.

About: Inference-time Objectives
Insights: Planning and reasoning to optimize inference-time objectives is important.

About: AI Regulation
Insights: Regulating applications, not R&D, is important.
