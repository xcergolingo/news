2023-04-22 AI news

news 2023/04/22

About: AGI
Insights: AGI was a philosophical and scientific concept ten years ago, but now it’s a marketing term. However, there has been no progress on the actual goal of achieving AGI.

About: AI
Insights: Traditional AI involves labeling, training, and predicting, while prompt-based AI involves prompting and predicting. Auto-regressive LLMs still can’t plan.

About: ChatGPT
Insights: ChatGPT can fall short in question answering in terms of comprehension, factualness, specificity, and inference. To enhance faithfulness, guidance for reasoning, association hints, and external data can be given. A preliminary study has been conducted to determine if ChatGPT is a good recommender.

About: Compression
Insights: LangChainAI has introduced multiple new methods to compress retrieved documents with respect to the query before passing to an LLM for generation.

About: Diffusion models
Insights: A data augmentation perspective on diffusion models and retrieval has been presented. Composer is a controllable diffusion model that has been trained on billions of (text, image) pairs. The model has 5 billion parameters.

About: Learning
Insights: Learning to Program with Natural Language involves asking LLM to learn natural language programs on training data from complex tasks and using the learned program to guide inference. LLMs Learning to Program w/ Natural Language has shown that on math, average performance outperforms direct zero-shot test performance by 18.3%.

About: Neural Radiance Fields
Insights: Nerfbusters has been developed to remove ghostly artifacts from casually captured NeRFs. Tetra-NeRF has been introduced to represent neural radiance fields using tetrahedra. Learning Neural Duplex Radiance Fields for Real-Time View Synthesis has been presented.

About: Open source models
Insights: With the release of open source models like StableML by StabilityAI, Dolly by Databricks, and Camel by Writer, there has been an increase in demand for running these models in LangChainAI. The revamped docs around Huggingface have been highlighted to show how easy this is.

About: Regulating Overfitting
Insights: Efficient Deep Reinforcement Learning Requires Regulating Overfitting.

About: Visual Prompting
Insights: Visual Prompting is being discussed as a way to take text prompting from NLP to vision. RSVP is available.

About: Adam Instability
Insights: A theory on Adam Instability in Large-Scale Machine Learning has been presented.

About: Sorcery
Insights: A video has been shared that shows a person performing a magic trick.
