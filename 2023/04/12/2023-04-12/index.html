news 2023/04/12

About: AI Gadfly
Insights: The AI Gadfly criticizes LeCun’s statement that solving AGI will lead to alignment coming along for free. The AI Gadfly suggests designing AI systems whose behavior optimizes objectives at inference time and fine-tuning objectives that align their behavior to human values.

About: BabyAGI
Insights: BabyAGI is a framework that allows an LLM to interact with the external world. With specialized prompt engineering and tool equipment, it can go a long way.

About: ChatGPT
Insights: ChatGPT can convert natural language instructions into executable robot actions. It outputs a sequence of actions, represents the operating environment formally, and infers updated state. Users can modify LLM prompt using natural language for robust robot operations.

About: Diffusion Models
Insights: Diffusion models can be harnessed for high-fidelity text-to-image synthesis. The spatial-temporal attention of diffusion models can be used for this purpose.

About: Image Processing
Insights: Caption-Anything is a versatile image processing tool that combines the capabilities of Segment Anything, Visual Captioning, and ChatGPT.

About: Inference
Insights: Inference with Reference achieves over 2x speed-up when there’s significant overlap between in-context reference and outputs exist. Prompt Pre-Training with Twenty-Thousand Classes for Open-Vocabulary Visual Recognition is another inference-related insight.

About: Language Models
Insights: OpenAGI is planning for AGI (Artificial General Intelligence). When LLM meets domain experts, it can lead to better insights. Why Think Step-by-Step? Using LLMs to Understand Reasoning explains how reasoning is effective when training data has clusters of variables influencing each other strongly. Learning to Tokenize for Generative Retrieval establishes the new state-of-the-art on the NQ320K dataset via better generative retrieval.

About: Neural Radiance Fields
Insights: Instance Neural Radiance Field and Neural Residual Radiance Fields for Streamably Free-Viewpoint Videos are two insights related to neural radiance fields.

About: WebBrain
Insights: WebBrain constructs a large-scale dataset WebBrain-Raw by extracting English wiki articles and their references.
