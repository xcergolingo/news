2023-04-28 AI news

news 2023/04/28

About: ChatGPT Prompt Engineering for Developers
Insights: OpenAI has launched a new course called ChatGPT Prompt Engineering for Developers, which is available for free. The course is created in collaboration with OpenAI and teaches developers how to build applications quickly using large language models (LLMs).

About: Deep Reinforcement Learning
Insights: Deep Reinforcement Learning (Deep RL) has been investigated for low-cost, miniature humanoid hardware in a dynamic environment. The method can synthesize sophisticated and safe agile soccer skills for a bipedal robot.

About: Human Values and Super-Intelligent Machines
Insights: Most people believe that super-intelligent machines of the future will have to be aligned with human values. However, getting it slightly wrong once will not spell doom on humanity. It is not as difficult as some experts make it out to be.

About: Metadata Fields and Vector Search
Insights: Vector search is good for capturing semantically similar texts, but often queries specify desired attributes like time, authorship, or other “metadata” fields, which vector search is not great at. SelfQueryRetriever has been introduced to address this issue.

About: Insects
Insights: Insects have outsmarted humans by a factor of 1000 in terms of the total number of neurons. However, there is no need to worry that they will kill all humans.

About: Bimanual Manipulation
Insights: Fine-grained bimanual manipulation can be learned with low-cost hardware.

About: TextDeformer
Insights: Geometry manipulation using text guidance is possible with TextDeformer.

About: Large-scale Language Models with Self-Controlled Memory System
Insights: The Self-Controlled Memory (SCM) system enables Large-scale Language Models (LLMs) to achieve multi-turn dialogue capabilities that are comparable to ChatGPT. SCM also enables LLMs to outperform ChatGPT in scenarios involving ultra-long document summarization or long-term conversations.

About: Superhuman AI Alignment
Insights: Worrying about superhuman AI alignment today is like worrying about turbojet engine safety in 1920. We do not have a working design for anything that could come close to becoming as smart as a dog, let alone a domestic robot that can clear the dinner table and fill up the dishwasher. Until we have a semi-credible design, we are discussing the sex of angels.

About: Constrained Text Generation
Insights: Distilled hidden Markov models can be used to control generations from LLMs. This approach achieves state-of-the-art performance on a challenging benchmark for constrained text generation, CommonGen.

About: Ray Conditioning
Insights: Ray Conditioning is a technique that trades photo-consistency for photo-realism in multi-view image generation.

About: GPT-3.5 and GPT-4
Insights: GPT-3.5 and GPT-4 have been evaluated for supporting real-world information needs in healthcare delivery.
