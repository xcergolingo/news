news 2023/04/23

About: LLMs (Language Model)
Insights:

  • LLMs are being fine-tuned to learn how to talk to APIs. The process involves crawling, GPT-4 data generation, and fine-tuning Vicuna-13B on synthetic data. LLM can then read new API docs and generate code.
  • Children acquire language primarily as an act, speaking is something they can do to influence their surroundings and achieve goals.
  • Children start using “no” a lot around age two, often reflexively. At some point, they learn to say “no” purposefully, which turns out to be an exceedingly effective strategy for shaping their environment to their liking.
  • Using adult-level psychology concepts to explain language acquisition in toddlers is not super relevant.

About: Bark
Insights:

  • Bark is a transformer-based text-to-audio model created by Suno. It can generate highly realistic, multilingual speech as well as other audio, including music, background noise, and simple sound effects.

About: Whisper JAX
