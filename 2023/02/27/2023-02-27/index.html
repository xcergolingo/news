2023-02-27 AI news

news 2023/02/27

About: LLMs, Insights: LLMs are exhibiting emergent behaviors at scale. Complexity science may have a resurgence. Chat-langchain app is completely reproducible in javascript.
About: LeNet, Insights: LeNet was trained on a Sun4 computer at Bell Labs.
About: LLaMA Team, Insights: Almost all the members of the LLaMA team are located at FAIR-Paris.
About: Prompt Tuning, Insights: Prompt Tuning (PT) helps LLMs by reducing variance when paired with natural language examples.
About: Bell Labs, Insights: At Bell Labs, employees don’t get famous by saving money.
About: Supervising LLMs, Insights: Making LLM reasoning states transparent is safer, but can be worse performance. Decomposing tasks and exposing LLM execution traces for human input can improve performance.
About: Open Agency Model of AI Alignment, Insights: AI control focuses on unitary agent model and deploying dominant agent to suppress unaligned AIs. Prompts generate proposals, human critics select, and agents implement tasks.
About: École Polytechnique and ENS, Insights: A number of members of the LLaMA team went through École Polytechnique and ENS, but not all. A big chunk went through the MVA (Master Vision Apprentissage at ENS Cachan).
About: Composer, Insights: Composer is a controllable diffusion model. It has been trained on billions of (text, image) pairs. The model has 5 billion parameters. Paper, Github link, and Project page are available.
About: Training LLMs, Insights: For actual downstream LLM applications, the question that matters more is “Given a fixed number of parameters or data, how should I train” (answer is usually: “train more”) rather than the Chinchilla question.
About: Linguistics, Insights: Linguistics is the scientific study of language.
About: Decentralized Training, Insights: Decentralized training will become a formidable force in open-source, large-scale AI developments.
About: Takeaways from LLMa Paper, Insights: LLMa paper suggests that large language models can be used to generate proposals, human critics select, and agents implement tasks.
About: Mini-Chain, Insights: Mini-Chain is a tiny library for large language models. It is a minimal implementation of prompt chains.
About: Training a Chatbot, Insights: A tutorial is available on how to train a chatbot on company’s documentation with LangChainAI in ten minutes.
About: Ghibli Diffusion, Insights: Ghibli Diffusion is a controllable diffusion model that can generate Ghibli-style illustrations.
About: AWS and Huggingface, Insights: AWS and Huggingface are working together to accelerate adoption of ML models.
