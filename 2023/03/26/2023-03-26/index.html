2023-03-26 AI news

news 2023/03/26

About: AIPlugins
Insights: LangChain has released an open-source attempt at using AIPlugins, which are the same ones ChatGPT is using. The community is excited to see what other techniques LangChainAI comes up with.

About: Code Generation
Insights: Reflection-Based GPT-4 Agent is state-of-the-art on code generation. It iteratively refines code, shifting the “accuracy bottleneck” from correct code generation to correct test generation. The model has been evaluated against other models, including GPT-4, CodeT, and PaLM, and has achieved the highest accuracy. Code and documentation are available.

About: Keras
Insights: TensorFlow 2.12 and Keras 2.12 were recently released, with many improvements in Keras, including a new native saving format and the new FeatureSpace all-in-one structured data preprocessing utility. Keras has had record-high download counts in the past few weeks.

About: Machine Learning
Insights: Hugging Face has released a new AppStore for machine learning called Hugging Face Spaces. It allows users to share and discover machine learning models, datasets, and other resources.

About: Text-to-Image Generation
Insights: Modelscope text2video generation and Text2Video-Zero are both text-to-image diffusion models that are zero-shot video generators. Localizing object-level shape variations with text-to-image diffusion models is also possible. P+ is an extended textual conditioning in text-to-image generation. Demos and project pages are available.

About: Transhumanism
Insights: The concept of transhumanism, which involves improving oneself and escaping the burden of the human condition, is not immoral.

About: General
Insights: Anything is worth doing if you do it with passion for the craft and compassion for those it will affect. Marketing and science do not mix well. Autoregressive models will still be widespread in five years. What you inherit is just your starting point, and you do not have to constrain yourself to it.
