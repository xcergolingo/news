2023-03-10 AI news

news 2023/03/10

About: ChatGPT
Insights:

  • ChatGPT has a system called Visual ChatGPT that incorporates different Visual Foundation Models to enable the user to interact with ChatGPT by sending and receiving not only languages.
  • ChatGPT rephrases each training sentence into multiple conceptually similar but semantically different sentences.
  • ChatGPT outperforms SoTA data augmentation methods for few-shot learning text classification.

About: COCO
Insights:

  • A method trained only on COCO achieves 23.4 PQ and 30.0 mIoU on the ADE20K dataset, with 8.3 PQ and 7.9 mIoU absolute improvement over the previous state-of-the-art.

About: DeepMind
Insights:

  • DeepMind has welcomed Clement to the team, who is one of the most talented people in AI, and they are thrilled to be building AGI together.
  • DeepMind has a paper on connecting pre-trained models (eg LLMs) to decisions via conditional generative modeling, reinforcement learning, optimal control, prompting, and planning. Models can drive decisions or serve world dynamics.

About: Diffusion Models
Insights:

  • TRACT improves FID by up to 2.4× on the same architecture for single-step diffusion and achieves new single-step Denoising Diffusion Implicit Models (DDIM) state-of-the-art FID.
  • Composer is a controllable diffusion model trained on billions of (text, image) pairs. The model has 5 billion parameters. Paper, Github link, and Project page are available.

About: Keras
Insights:

  • Keras is like a cheat code for ML devs. Use this one trick to shrink your codebase by 2x and train your models 25% faster.

About: Large Language Model
Insights:

  • The large model singularity suggests that time might stop before we get to the year 2030, which would require models with an infinite number of parameters.
  • Optimizing for lower-cost LLM generation can significantly affect text generation costs. Experiments with GPT-3.5 verify effectiveness.
  • Large Language Model API has a streaming API in a Gradio demo. Try out models on Hugging Face model hub with streaming API.

About: OpenXLA Project
Insights:

  • Google is excited to collaborate on the OpenXLA Project to foster great AI tooling (great ML performance, framework compatibility, highly configurable, and more).

About: Video-P2P
Insights:

  • Video-P2P works well on real-world videos for generating new characters while optimally preserving their original poses and scenes.

About: X-Avatar
Insights:

  • X-Avatar is a project for expressive human avatars.
