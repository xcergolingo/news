About: Composer, Insights: Composer is a controllable diffusion model.It has been trained on billions of (text, image) pairs. The model has 5 billion parameters. Paper, Github link, and Project page are available.

About: GPT-4, Insights: GPT-4, aka “DV”, is a leaked “Foundry” platform from OpenAI. Most likely use cases for vertical LLM providers, pricing hints at an unpublished algorithmic breakthrough in LLMs, and more are discussed in the long-form commentary. Link is available.

About: LLM Overconfidence & Uncertainty, Insights: Q&A accuracy suffers when LLMs are prompted to use language like “I’m 100% certain”. GPT-3 has poor calibration in certainty expressions, but expressions of uncertainty might lead to better model calibration. Paper link is available.

About: Internet Explorer, Insights: Internet Explorer explores the web in a self-supervised manner to progressively find relevant examples that improve performance on a target dataset. Project page and Abstract link are available.

About: Online Agent Learning on the Open Web, Insights: Online Agent Learning on the Open Web treats internet as open-ended, always-updating dataset. Self-supervised exploring finds relevant data to quickly improve small model on desired task. Paper and Code link are available.

About: Prompt Templates, Insights: Prompt Templates are a key building block, and can be used to easily inject up-to-date information in prompts.

About: ML Pipelines, Insights: ML pipelines involve many consecutive steps of semantics-aware data preprocessing and feature engineering, which is labor-intensive and brittle. DL replaces all this with a single differentiable function trained with gradient descent.

About: Directed Diffusion, Insights: Directed Diffusion is a direct control of object placement through attention guidance. Abstract and Project page link are available.

About: Deep Learning Limitations, Insights: Deep learning has stark limitations, which means that it’s not a good fit for every problem. But it didn’t reach its current level of popularity simply because of hype. DL really has fundamental properties that are remarkably effective.

About: ELITE, Insights: ELITE is a novel learning-based encoder for fast and accurate customized text-to-image synthesis. Project page and Abstract link are available.

About: Differentially Private Diffusion Models, Insights: Differentially Private Diffusion Models generate useful synthetic images by privately fine-tuning ImageNet pre-trained diffusion models with more than 80M parameters. Abstract link is available.
