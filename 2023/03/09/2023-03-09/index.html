About: ChatGPT
Insights:

  • ChatGPT release has rethought memory in @LangChainAI, allowing memory objects to return List[ChatMessage] (rather than strings) and made memory more modular so it can be used outside of chains.
  • One common pain point is that ChatGPT does not work amazingly with preconfigured @LangChainAI chains because of hardcoded default prompts that are optimized for GPT3. The solution is ✨Prompt Selectors✨.

About: Larger LLMs
Insights:

  • Larger LLMs do in-context learning differently. Overriding semantic priors is an emergent ability of scale. When ICL labels are semantically unrelated to inputs (foo/bar vs neg/pos), bigger LLMs better learn mappings. Instruction-tuning helps both. Paper is available.
  • Larger language models do in-context learning differently. Studies how in-context learning (ICL) in LMs is affected by semantic priors versus input–label mappings.
