news 2023/03/11

About: GigaGAN
Insights: GigaGAN is a text-to-image synthesis model with 1B parameters. It generates high-resolution images at a faster speed than other models such as Stable Diffusion v1.5, DALL·E 2, and Parti-750M. The model can synthesize 16-megapixel pixels in 3.66 seconds.

About: ZoeDepth
Insights: ZoeDepth is a zero-shot transfer model that combines relative and metric depth. It has a Gradio demo available on Huggingface.

About: Cones
Insights: Cones is a diffusion model for customized generation with concept neurons. The paper and abstract are available.

About: MathPrompter
Insights: MathPrompter is a mathematical reasoning model that uses large language models. It improves over the state-of-the-art on the MultiArith dataset and can generate multiple algebraic expressions or Python functions to solve the same math problem in different ways. The paper is available.

About: LLM Prompt Pattern Catalog
Insights: The LLM Prompt Pattern Catalog provides prompt patterns for automating software development, including output customization, prompt improvement, error identification, input semantics, context control, and interaction. The patterns significantly enrich ChatGPT’s capabilities. The paper is available.

About: Planning with Large Language Models for Code Generation
Insights: The Planning with Large Language Models for Code Generation framework can generate programs that consistently achieve higher scores on public coding challenge benchmarks. The project page and GitHub link are available.

About: PAC-NeRF
Insights: PAC-NeRF is a physics-augmented continuum neural radiance fields model for geometry-agnostic system identification. The abstract, project page, and GitHub link are available.

About: Open-world Instance Segmentation
Insights: Open-world Instance Segmentation is a top-down learning model with bottom-up supervision. The abstract and project page are available.

About: 3D Video Loops from Asynchronous Input
Insights: 3D Video Loops from Asynchronous Input is a model that generates 3D video loops from asynchronous input. The abstract and project page are available.

About: ELITE
Insights: ELITE is a model available on Huggingface with a Gradio demo. No further details are given.

About: 3DGen
Insights: 3DGen is a triplane latent diffusion model for textured mesh generation. The abstract is available.

About: Cherry-Picking with Reinforcement Learning
Insights: Cherry-Picking with Reinforcement Learning is a model that performs superhuman-level cherrypicking with RL. The project page and abstract are available.
