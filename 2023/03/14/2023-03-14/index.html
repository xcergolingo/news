2023-03-14 AI news

news 2023/03/14

About: ChatGPT

  • Open-Source ChatGPT Replication has been released, which includes a smaller LLM fine-tuned to filter which questions the bot responds to, and an extensible retrieval system to augment bot responses with info from external data.
  • Rewarding Chatbots for Real-World Engagement with Millions of Users has resulted in more than 30% increase in user retention for a GPT-J 6B model. The model and code have been open-sourced.

About: SVB (Silicon Valley Bank)

  • The collapse of SVB could lead to large layoffs, damage to trust in banks, and setting back AI & tech innovation by years. It is important to protect depositors who had taken no reckless risks.

About: AI & startup community

  • The AI & startup community has shown the best of itself in dealing with SVB’s collapse. People are helping each other, sharing tips, and even wiring funds on trust and without docs so someone else can make payroll.

About: Deep learning

  • Deep learning based super resolution is being used, which raises the question of at what point does it become “replacement” instead of “enhancement” of the original.

About: Keras

  • Keras APIs are available to make programs globally deterministic, which sets the Python seed, NumPy seed, and Keras seed simultaneously.

About: NeuralLift-360

  • NeuralLift-360 is a project that lifts an in-the-wild 2D photo to a 3D object with 360° views.

About: Originating ideas

  • In a world where endless remixing is effortless, the ability to originate ideas will be increasingly precious.

About: Rail

  • A new formatting language called Rail has been developed for constructing prompts.

About: StyleGANEX

  • StyleGANEX is a project that allows for StyleGAN-based manipulation beyond cropped aligned faces.

About: Tag2Text

  • Tag2Text is a project that guides vision-language models via image tagging.

About: Visual ChatGPT

  • Visual ChatGPT is a project that allows for talking, drawing, and editing with visual foundation models.

About: MVImgNet

  • MVImgNet is a large-scale dataset of multi-view images.

About: Reliable AI markup Language

  • Reliable AI markup Language adds structure, type, and quality guarantees to LLMs. It includes Pydantic-style validation of LLM output, enforces structure and type guarantees, and is a dialect of XML.
