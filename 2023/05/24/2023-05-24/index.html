news 2023/05/24

About: MMS (Massively Multilingual Speech)
Insights: MMS can do speech2text and text speech in 1100 languages. It can recognize 4000 spoken languages. The code and models are available under the CC-BY-NC 4.0 license. MMS has half the word error rate of Whisper. The project increases the number of supported languages by 10-40x, depending on the task. The main ingredients are a new dataset based on readings of publicly available religious texts.

About: RWKV (Reinventing RNNs for the Transformer Era)
Insights: RWKV proposes a novel model architecture that combines the efficient parallelizable training of Transformers with the efficient inference of RNNs. The approach leverages a linear attention mechanism.

About: LLM (Large Language Models)
Insights: Prompting tests metalinguistic judgment. Other methods directly read out probabilities over strings. Metalinguistic judgments are inferior. So negative result from prompt is not evidence LLM lacks a competence. In-context learning abilities can be recursively improved via in-context learning itself. Meta-ICL adaptively reshapes LLM priors & modifies learning strategies. On real-world regression, it is competitive with traditional algorithms. RecurrentGPT is a language-based simulacrum of the recurrence mechanism in RNNs. RecurrentGPT is built upon a large language model (LLM) such as ChatGPT and uses natural language to simulate the Long Short-Term Memory. LLM vs LLM: Detecting Errors via Cross Examining Agents. An incorrect claim is likely to result in inconsistency with other claims. Multi-turn interactions between LLM that generated claim and Examiner LLM. Outperforms baselines across factual benchmarks. To Repeat or Not To Repeat: Studies what would happen if we train LLM with repeated data and how we can alleviate the LLM mult-epoch degradation.

About: AI Game Jam
Insights: Introducing the Open Source AI Game Jam. Let’s use AI tools to unlock creativity and make a game in a weekend. It is open to everyone and free to participate.

About: Diffusion Models
Insights: Investigate reinforcement learning methods for directly optimizing diffusion models for such objectives. We describe how posing denoising as a multi-step decision-making problem enables a class of policy gradient algorithms.

About: ControlVideo
Insights: ControlVideo is adapted from ControlNet, leverages coarsely structural consistency from input motion sequences, and introduces three modules to improve video generation. Firstly, to ensure appearance coherence.

About: CodeCompose
Insights: CodeCompose is an AI-assisted code authoring tool developed and deployed at Meta internally. CodeCompose is based on the InCoder LLM that merges generative capabilities with bi-directionality.

About: VideoLLM
Insights: VideoLLM leverages the sequence reasoning capabilities of pre-trained LLMs from NLP for video sequence understanding.

About: Learning Visual Correspondences
Insights: Our newest work towards solving one of the most fundamental problems of perception - learning visual correspondences using video data. Very simple idea with promising results outperforming multiple state-of-the-art results.

About: Professionals Skillfully Crafting or Fixing Things
Insights: My favorite kind of YouTube video is professionals skillfully crafting or fixing things. Love watching a job well done and learning how things are made.
