news 2023/05/19

About: LLM base models
Insights: There is a debate around whether LLM base models should be closed or open. Meta argues for openness, starting with the release of LLaMA (for non-commercial use), while OpenAI and Google want to keep things closed and proprietary.

About: LLM Agents
Insights: LLM Agents can autonomously improve in negotiation from AI criticism. Two LLMs negotiate as buyer/seller bargain, while a third LLM provides feedback to improve negotiation strategies. Previous negotiation history + AI feedback for in-context learning improves performance.

About: PaLM 2
Insights: A technical report on PaLM 2 is available. No further details are given.

About: Verbalize Story Videos
Insights: A video is worth 4096 tokens: Verbalize Story Videos to Understand Them in Zero Shot. Paper page and project page are available.

About: FastComposer
Insights: FastComposer generates images of multiple unseen individuals with different styles, actions, and contexts. It achieves 300 times-2500 times speedup compared to fine-tuning-based methods. Paper page and project page are available.

About: LangChain Retrieval
Insights: Retrieval is important. Vector similarity search is not enough. LangChain Retrieval Webinar will discuss this topic with two experts.

About: Task Recognition and Learning
Insights: What LLMs Learn During In-Context Learning. Task recognition (TR) through demos w/out ground-truth labels. Task learning (TL) captures new input-label mappings unseen in pre-training. TR does not further improve w/ LLM size. TL is acquired as LLM scales.

About: Text-Image Alignment Evaluation
Insights: What You See is What You Read? Improving Text-Image Alignment Evaluation. Paper page is available.

About: NerfBridge

About: SLiC-HF
Insights: SLiC-HF presents a competitive alternative to the PPO RLHF while being much simpler to implement, easier to tune, and more computationally efficient in practice.

About: GPT-Bargaining

About: DoReMi
Insights: DoReMi optimizes data mixtures speeds up language model pretraining. In experiments, use DoReMi on a 280M-parameter proxy model to find domain weights for training an 8B-parameter model (30x larger) more efficiently. On The Pile, DoReMi improves perplexity across all domains. Paper page and Github link are available.
