news 2023/05/17

About: AI Doomerism, Insights: Canadian writer Stephen Marche wrote an opinion piece in The Guardian against AI doomerism. He argues that AI will not replace all jobs and that we should not fear it.

About: AutoRecon, Insights: AutoRecon is an automated 3D object discovery and reconstruction system. It has an abstract and a paper page, as well as a project page.

About: CodeT5+, Insights: CodeT5+ is an instruction-tuned large language model for code understanding and generation. It achieved a new state-of-the-art result on the HumanEval code generation task.

About: DarkBERT, Insights: DarkBERT is a language model for the dark side of the internet. It has an abstract and a link to the paper.

About: Dr. LLaMA, Insights: Dr. LLaMA is a method for improving small language models in domain-specific QA via generative data augmentation. It has an abstract and a link to the paper page.

About: GPT-Sentinel, Insights: GPT-Sentinel is a system for distinguishing human and chatbot-generated content. It has an abstract and a link to the paper page.

About: In-Context Learning, Insights: In-context learning is a method for improving language models by incorporating small models as plug-ins. It beats state-of-the-art fine-tuned models while addressing the instability of large language models.

About: Jobs, Insights: Machines have taken over many jobs in the 21st century that were once occupied by people in the 19th century. Most 21st century jobs would be incomprehensible to a 19th century farmer.

About: ML Observability, Insights: ArizeAI has released an open-source library for LLMs and deep NNs more generally. It provides visual clustering analysis and model interpretability in latent space.

About: Small Models, Insights: Small models are valuable plug-ins for large language models. There is an abstract and a link to the paper page.

About: Symbol Tuning, Insights: Symbol tuning is a method for finetuning language models on in-context input-label pairs where natural language labels are replaced with arbitrary symbols.

About: Tech Doomerism, Insights: Tech doomerism is a species of hype.

About: TinyStories, Insights: TinyStories is a method for training and evaluating language models that are much smaller than state-of-the-art models or have much simpler architectures.

About: Vaccination, Insights: Covid vaccination is a huge achievement that has saved tens of millions of lives.

About: Vision-Language Pre-Training, Insights: Improved baselines for vision-language pre-training have been developed. There is an abstract and a link to the paper page.

About: Scaling LLM Interpretability, Insights: A gradient descent method grounded in the theory of causal abstraction has uncovered perfect alignment between interpretable symbolic algorithms and small deep learning models. Scaling this alignment search to LLMs helps understand Alpaca-7B.
