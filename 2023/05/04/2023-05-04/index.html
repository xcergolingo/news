2023-05-04 AI news

news 2023/05/04

About: ChatGPT

  • “Is Your Code Generated by ChatGPT Really Correct?” proposes EvalPlus – a code synthesis benchmarking framework to rigorously evaluate the functional correctness of LLM-synthesized code.
  • “As the cofounder and chair of the ICLR Foundation, and despite being post-PhD with more than 10 papers, I am very much in favor of letting authors use the tools they deem most useful to help them write technical papers, including LLMs.”

About: Physically-simulated tennis players

  • “Learning Physically Simulated Tennis Skills from Broadcast Videos” demonstrates a system that produces controllers for physically-simulated tennis players that can hit the incoming ball to target positions accurately using a diverse array of strokes.

About: Long-Range Transformers

  • “Unlimiformer: Long-Range Transformers with Unlimited Length Input” improves pretrained models such as BART and Longformer by extending them to unlimited inputs without additional learned weights and without modifying their code.
  • “Unlimiformer: Long-Range Transformers with Unlimited Length Input” demonstrates Unlimiformer’s efficacy on several long-document and multi-document summarization benchmarks, showing that it can summarize even 350k token-long inputs from the BookSum dataset, without any input.

About: Large Language Models (LLMs)

  • “Is Your Code Generated by ChatGPT Really Correct?” extensive evaluation across 14 popular LLMs (including GPT-4 and ChatGPT) demonstrates that HUMANEVAL+ is able to catch significant amounts of previously undetected.
  • “Mass-Editing the Memory of an LLM” LLMs have no built-in way update knowledge. Previous SoTA updated single associations at a time. New direct editing method can scale up to perform thousands of memory edits at once.
  • “LLMs & Causal Reasoning” LLMs establish new SoTA on multiple causal benchmarks.
  • “The Internal State of an LLM Knows When its Lying” creates a dataset of true & false statements across 6 topics. Trains classifier to detect which statement is true or false based on an LLM’s activation values. Significantly outperforms in detecting veracity.
  • “Distill or Annotate? How to Most Efficiently Use a Fixed LLM Budget” extensive experiments on 6 diverse NLP tasks. Distilling LLM (11B) to small language model (60M) is more cost-effective. Compared to annotating more data to directly train small model.
  • “Poisoning LLMs During Instruction Tuning Is Not Hard” adversaries can poison data to manipulate. With only 100 poison examples: can cause arbitrary phrases to induce degenerate outputs across hundreds of held-out tasks. Larger LLMs increasingly vulnerable.

About: Text-to-Image Generation

  • “Pick-a-Pic: An Open Dataset of User Preferences for Text-to-Image Generation” is an open dataset of user preferences for text-to-image generation.
  • “Key-Locked Rank One Editing for Text-to-Image Personalization” presents Perfusion, a new text-to-image personalization method. With only a 100KB model size, Perfusion can creatively portray personalized objects. It allows significant changes in their appearance, while maintaining.
  • “TMR: Text-to-Motion Retrieval Using Contrastive 3D Human Motion Synthesis” extensive experiments on the KIT-ML and HumanML3D datasets show that TMR outperforms the prior work by a significant margin, for example reducing the median rank from 54 to 19.

About: AI Tools

  • “Structured Tools” introduces new tools in @LangChainAI that can now take multiple inputs. It also introduces a new agent type for these tools.

About: Mojo

  • “Nice demo of Mojo, a new Python-compatible language with a parallelizing compiler that can import Python libraries.”
