About: AI model or technique
Insights: There is currently no AI model or technique that could represent an extinction risk for humanity. This is according to a tweet that clarifies that even in the future, there is no such risk.

About: ByteFormer
Insights: ByteFormer is a transformer backbone that achieves an ImageNet Top-1 classification accuracy of 77.33% when training and testing directly on TIFF file bytes. This is according to a tweet that shares a link to the paper page.

About: Diffusion models
Insights: Self-guidance is a method that provides greater control over generated images by guiding the internal representations of diffusion models. This is according to a tweet that introduces the method and demonstrates its properties.

About: GPT
Insights: GPT-4 knows a lot about the world, including factual tasks such as location, distance, and elevation, as well as more complex tasks such as generating country outlines, travel networks, and supply chain analysis. This is according to a tweet that shares an experiment on how an LLM sees the world’s geography.

About: Hiera
Insights: Hiera is a hierarchical vision transformer without the bells-and-whistles. It does not have vision-specific components, but it still leads to supervised classification performance. This is according to a tweet that shares a link to the abstract.

About: LLaVA-Med
Insights: LLaVA-Med is a vision-language conversational assistant that can answer open-ended research questions of biomedical images. It is trained in one day using a cost-efficient approach. This is according to a tweet that shares a link to the paper page.

About: Memory Viewpoint
Insights: Large language models based on transformers have achieved great empirical successes. However, as they are deployed more widely, there is a growing need to better understand their internal mechanisms. This is according to a tweet that shares a link to the paper page.

About: SnapFusion
Insights: SnapFusion is a generic approach that unlocks running text-to-image diffusion models on mobile devices in less than 2 seconds. This is according to a tweet that shares a link to the paper page.

About: StyleDrop
Insights: StyleDrop is a method that enables the synthesis of images that faithfully follow a specific style using a text-to-image model. The proposed method is extremely versatile and captures nuances and details of a user-provided style. This is according to a tweet that introduces the method.
